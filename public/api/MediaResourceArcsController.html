MediaResourceArcs

Media resource arcs are the relations between media resources. Like a set has multiple media entries and a media entry can have multiple parent sets (Graph).

Get media resource arcs:

/media_resource_arcs [GET]

Required Parameters

  • parent_id Integer The id of the parent media resource (set).

Optional Parameters

  • child_id Integer The id of the child media resource (set/entry).

Response Fields

  • parent_id Integer The id of the parent media resource.
  • child_id Integer The id of the child media resource.
  • highlight Boolean A status indicator if the arc is highlighted or not.

Examples

Request the media resources arcs for the media set with the id 1. Request {"parent_id":1} Response {"media_resource_arcs": [{"parent_id": 1, "child_id": 23, "highlight": false}, {"parent_id": 1, "child_id": 12, "highlight": true}]} Returns two media resources arcs one is highlighted the other isnt.
Request the media resources arcs for the media set with the id 1. Requesting through plain urls is also possible (/media_resource_arcs/:parent_id). Request /media_resource_arcs/1 Response {"media_resource_arcs": [{"parent_id": 1, "child_id": 23, "highlight": false}, {"parent_id": 1, "child_id": 12, "highlight": true}]} Returns two media resources arcs one is highlighted the other isnt.

Update media resource arcs:

/media_resource_arcs [PUT]

Required Parameters

  • media_resources_arcs Array The collection of media resources arcs which should be updated.
  • media_resources_arcs[].parent_id Integer The parent id of the media_resource_arc.
  • media_resources_arcs[].child_id Integer The child id of the media_resource_arc.
  • media_resources_arcs[].highlight Boolean The highlight status of that media resource arc.

Optional Parameters

Response Fields

Examples

Set the highlight status for the arc between parent id 1 and child id 23 to true. Request {"media_resource_arcs": [{"parent_id": 1, "child_id": 23, "higlight": true}]} Response {} Returns a Status: 200 (empty Hash).
